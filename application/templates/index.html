A design code library

This prototype is intended to show how data found on the Planning Data Platform can be easily presented in different ways on other websites, maps and services, such as a design code libary curated by the Office for Place.

  • Design code library
  • Design code areas
  • Map of design code areas

Data standard

We are currently developing a data standard to help local planning authorities to list their design codes in a way which allows them to be easily collected into national datasets.

  • Planning consideration
  • Draft specification
  • Discussion

Data

This prototype uses data we gathered to help test the proposed data standard and which can be found on the Planning Data Platform:

  • List of design codes
  • List of design code areas
  • National map of design code areas

The data is available through the API or downloaded in a number of different formats:

  • Design code dataset
  • Design code area dataset
  • Design code status dataset
  • Design code category dataset

Next steps

The data we currently have on the platform was collected some time ago and is held on GitHub collection. This doesn't yet include all of the codes listed in March 2023.

Not all the codes we have listed have public documents, geospatial boundaries, and we haven't tagged them with categories or assigned their status.

Our next steps are to work with the Office of Place to further develop the data standard, and with LPAs to help them provide data to the standard.
